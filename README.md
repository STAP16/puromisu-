## FastAPI, React JS, Docker Compose, Nginx, PosgreSQL, Async, React-Router

## Description
## Приложение полностью асинхронная, функционал простой - Регистрация, Аутентификация Через JWT токены, которые хранятся в куках
## Суть проекта:
## Деплой, JWT, цикл регистрации и авторизации, хранение access токена в куках, refresh токен так-же хранится в куках
## Маршрутизациия, защищенные маршруты, Auth Provider
## Async взаимодействие с PosgreSQL, Паттерн - Provider / Services
## Время жизни токенов 15 минут, 7 дней
## GPT использовался в качестве дебага и проверки на наличие ошибок, связанных с развертыванием приложения
